My Projects

YuMi Vision with Dice

This is a project I built for our product center. It uses an ABB YuMi Cobot with cognex vision and a FlexLoader. The user selects a number on the HMI and then the FlexLoader presents the dice to the camera. The vision algorithim in the camera sends all of the recognized matches with coordinates in an array to the Cobot. The Cobot will then pick up all of dice that matches the selected number.

Lawn Mowers

This is a full system build I spearheaded for Husqvarna. I programmed 10 production ready robots for lawn mower assembly. It included 5 press tending robots, 3 Deburring robots with air compensation, 1 3D vision inspection for QC, and 1 palletizing robot. The video was taken at MOD build, before shipping the line to the customer.

3D Vision Bin Picking

This robot is using a 3D vision system running on ROS. The vision algorithim was setup by another team, but I was responsible for the installation, calibration, and safety logic. The 3D camera scans the chute, then sends over the target information with 6 DOF over to the robot. The robot gets the package from the chute, places it on the conveyor, and the package gets scanned and sent out for processing.

YuMi Cobot with AGV

This was another great project where my team created something ahead of its time. I programmed a Cobot on an AGV to go to pick and place parts from different locations determined by the queuing system. My colleague was responsible for mapping the environment and queuing system using python. I setup the vision inspection and coordinated motion control. The robot traveled to its location using Lidar to avoid collisions, then took a picture of a QR code, which it used to setup the work object frame used for the vision system. The Naruto run is a good laugh, but really helps to avoid collisions with other Cobots. The rest of the video is cut out for NDA reasons.

YuMi Espresso

This project I completed showcases the YuMi's precise motion and cobot capabilities. The user grabs a coffee cup and places it on the stand. Underneath the cup, there are differnt stickers that correlate with the different espresso types. The YuMi picks up the cup and takes a picture of the sticker and uses the pattern algorithim to select the espresso type (decaf, bold, caramel). Then YuMi gets the espresso pod, based of the pattern recognized and then starts the automatic brewing process.
